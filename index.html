⚔ Bucle WHILE: Cazando al Kraken

Historia: ¡Un terrible Kraken está atacando nuestro barco! No sabemos cuántos cañonazos necesitaremos para derrotarlo, pero debemos seguir disparando MIENTRAS tenga vida. ¡El bucle WHILE es perfecto para esto!

🦑 Batalla contra el Kraken
# El Kraken empieza con 100 puntos de vida vida_kraken = 100 disparos = 0 print("🦑 ¡El Kraken aparece! Vida: 100") print("💣 ¡Preparar los cañones!") # Seguimos disparando MIENTRAS el Kraken tenga vida while vida_kraken > 0: disparos += 1 daño = 25 # Cada cañonazo hace 25 de daño vida_kraken -= daño print(f"💥 ¡BOOM! Disparo #{disparos}") print(f"🦑 Vida del Kraken: {vida_kraken}") if vida_kraken > 0: print("😤 ¡El Kraken sigue luchando!") else: print("🎉 ¡VICTORIA! ¡El Kraken ha sido derrotado!")

⚔️ ¡A la batalla!

# Buscamos tesoros hasta encontrar el cofre dorado tesoro_encontrado = False intentos = 0 print("🔍 Buscando el cofre dorado...") while not tesoro_encontrado: intentos += 1 print(f"🕳️ Excavando en el lugar #{intentos}...") # Simulamos encontrar el tesoro después de algunos intentos if intentos == 3: tesoro_encontrado = True print("💎 ¡EUREKA! ¡Cofre dorado encontrado!") else: print("🪨 Solo tierra y rocas... sigamos buscando") print(f"🎯 Misión completada en {intentos} intentos!")
¡La búsqueda del tesoro está por comenzar!
⚔ ¡Practica bucles WHILE!
👆 ¡Cuidado con los bucles infinitos! Siempre cambia la condición
⚔ Pistas para WHILE:
• Siempre cambia la variable de control dentro del bucle
• contador += 1 es lo mismo que contador = contador + 1
• La condición se evalúa ANTES de cada iteración
• ¡Ten cuidado con los bucles infinitos!

⚖️ FOR vs WHILE: ¿Cuándo usar cada uno?

Aspecto 🗺️ Bucle FOR ⚔ Bucle WHILE
¿Cuándo usar? Cuando SABES cuántas veces repetir Cuando NO SABES cuántas veces repetir
Ejemplo Pirata Visitar 5 islas del mapa Luchar hasta derrotar al Kraken
Sintaxis for item in lista: while condicion:
Control Automático Manual (debes cambiar la condición)

💡 Consejo del Capitán CodeBeard:

FOR: "Usa FOR cuando tengas una lista de cosas que hacer o un número fijo de repeticiones."

WHILE: "Usa WHILE cuando necesites repetir algo hasta que se cumpla una condición especial."
