🧭 Resumen del Bucle FOR

¿Cuándo usar FOR? Cuando sabes exactamente cuántas veces quieres repetir algo:

  • Recorrer una lista de elementos (islas, tripulantes, tesoros)
  • Repetir una acción un número específico de veces
  • Procesar cada elemento de una colección

Sintaxis básica:

for elemento in lista:
    # código a repetir
    print(elemento)
